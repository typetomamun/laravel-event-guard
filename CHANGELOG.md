# Changelog

All notable changes to `laravel-event-guard` will be documented in this file.

## 1.0.0 - 2024-01-01

### Added
- Initial release
- Multi-tenant permission system
- Event-scoped roles and permissions
- Blade directives for permission checks
- Middleware for route protection
- Automatic owner detection
- Support for Laravel 11 and 12
- Complete test suite
- Comprehensive documentation

### Features
- Event types: Shop, Forum, Group, Announcement
- Role management per event
- Permission management per event
- Owner bypass for all permissions
- Helper functions
- Seeders for event types
